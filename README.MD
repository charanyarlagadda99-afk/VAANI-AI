# VAANI — Voice Adaptive AI for National Emergency Intelligence

## 🚨 Problem
Emergency response systems fail due to language barriers, delayed response, and call overload.

## 💡 Solution
VAANI is a real-time AI voice triage system that:
- Handles emergency calls instantly
- Works in multiple Indian languages
- Extracts critical data in real-time
- Responds with ultra-low latency voice AI

## ⚙️ Tech Stack
- Murf AI (Falcon TTS)
- LiveKit (Telephony + Voice Agents)
- Groq / LLM APIs
- Python (Backend)
- WebRTC

## 🔥 Features
- Real-time voice conversations
- Multilingual support (Hindi, Telugu, Kannada, English)
- AI-based emergency classification
- Scalable architecture
